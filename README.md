# Xfel::Timew::Jira

This is a report and sync tool for TimeWarrior -> Jira ticket's worklog.

## Installation

```bash
gem install xfel-timew-jira

echo "require 'xfel-timew-jira'" > ./path/to/timew/extensions/xfel-timew-jira.rb
```

# Usage

```bash
timew xfel-timew-jira
```
